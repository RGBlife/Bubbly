## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/RGBlife/bubbly.git
    ```

2. Navigate to the project directory:
    ```bash
    cd bubbly
    ```

3. Install dependencies:
    ```bash
    npm install
    ```

## Usage

1. Start the server:
    ```bash
    npm start
    ```

2. Open your browser and visit:
    ```
    http://localhost:3000
    ```

3. Begin engaging in real-time conversations!
